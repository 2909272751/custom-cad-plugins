# xrefpick

AutoCAD 外部参照选择过滤和外参图层临时改色插件。

这个插件用于把 DWG 外部参照当作“底图”处理：尽量避免误选外参，同时可以把外参依赖图层临时改成指定色号，方便看图，之后一键恢复原颜色。

## 命令

| 命令 | 作用 |
| --- | --- |
| `XREFPICK` | 打开控制菜单：切换过滤、外参改色、恢复颜色、查看状态 |
| `XREFPICKLOG` | 显示诊断日志和配置文件路径 |

## 使用流程

1. 下载 `xrefpick-v0.1.7-autocad2021.zip` 并解压。
2. 运行解压目录里的 `unblock.ps1`。
3. 在 AutoCAD 命令行输入 `NETLOAD`。
4. 加载 DLL：

```text
xrefpick-v0.1.7-autocad2021.dll
```

5. 输入命令：

```text
XREFPICK
```

6. 插件会显示当前状态，然后让你选择操作：

```text
当前状态：
选择过滤：已开启
外参改色：未改色或无恢复记录
日志路径：%TEMP%\XREFPICK.log
配置路径：%APPDATA%\custom-cad-plugins\xrefpick.cfg
请选择操作 [切换过滤(F)/隐藏填充(H)/恢复填充(U)/外参改色(C)/恢复颜色(R)/状态(S)] <F>:
```

## 操作说明

### 切换过滤 F

开启后，插件会尽量把外部参照从选择里排除：

- 点击、框选、交叉框选加入选择集时会自动移除外参。
- 点击、框选、交叉框选加入选择集时也会自动移除锁定图层上的对象。
- 命令开始前会再次清理当前选择集。
- 鼠标悬停命中外参时会轻量检查并刷新，尽量减少外参高亮残留。
- 自动增强 `PICKAUTO`，允许从外参或填充对象上直接拖出窗口/交叉框选。
- 自动增强 `PICKDRAG`，同时支持拖框和两点框选。
- 临时关闭 `PICKSTYLE` 的填充关联选择，减少填充区域抢走框选起点。

AutoCAD 的底层命中测试仍可能先碰到外参，所以极少数情况下仍会有一瞬间反馈；插件会尽量马上清掉。

### 外参改色 C

把当前图纸里的外参依赖图层改成指定色号。

```text
请输入外部参照图层目标色号 <35>:
将所有外部参照图层颜色改为 35，是否继续 [是(Y)/否(N)] <Y>:
```

说明：

- 色号可输入 1 到 255，默认 35。
- 只改当前图纸里的外参图层覆盖，不修改外参 DWG 源文件。
- 原颜色记录保存到当前 DWG，方便后续恢复。
- 对象如果不是 ByLayer 颜色，可能不会跟随图层颜色变化。

### 恢复颜色 R

把上次记录的外参图层颜色恢复回来。

```text
检测到 86 个外参图层颜色记录，是否恢复 [是(Y)/否(N)] <Y>:
```

恢复完成后会清除当前 DWG 里的颜色记录。

### 隐藏填充 H / 恢复填充 U

只对外参和锁定普通块相关填充生效，减少 Hatch/填充区域抢走框选起点。

实现方式：

- 外参：隐藏外参依赖填充图层，不影响本图同名图层。
- 锁定普通块：隐藏块定义内 Hatch/Solid 对象，并记录对象可见性用于恢复。
- `U` 会恢复隐藏的外参填充图层和锁定块填充对象。
- 不删除 Hatch 对象，不修改外参源文件，不使用全图 `FILLMODE=0`。

### 状态 S

只显示当前过滤状态、外参改色记录数量、日志路径和配置路径。

## 识别规则

- 识别直接选中的 DWG 外部参照块引用。
- 识别覆盖参照。
- 如果选中的对象属于外部参照块记录，也会从选择集中移除。
- 外参图层通过图层依赖状态或图层名里的 `|` 判断，例如 `底图|WALL`。

## 日志

如果结果不对，运行：

```text
XREFPICKLOG
```

日志默认写到：

```text
%TEMP%\XREFPICK.log
```

过滤开关配置默认写到：

```text
%APPDATA%\custom-cad-plugins\xrefpick.cfg
```

## 构建

默认按本机 AutoCAD 2021 路径构建：

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1
```

指定 AutoCAD 安装目录：

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1 -AcadPath "D:\autocad\AutoCAD 2021" -AcadLabel "autocad2021"
```

输出：

```text
dist\xrefpick-v0.1.7-autocad2021.dll
```
