# LayerOffsetPlugin v0.1.0

首个版本。

## 新增

- 新增 `LOFFSET` 命令。
- 支持点选对象确定目标图层。
- 支持高亮目标图层对象，并在进入下一步前确认。
- 支持框选/交叉框选需要 offset 的区域，只处理框选范围内属于目标图层的对象。
- 支持输入 offset 距离。
- 支持选择方向：`N` 为内，`W` 为外，默认 `W`。
- 支持选择是否删除原图形，默认不删除。
- 支持对 `Polyline`、`Line`、`Arc`、`Circle`、`Ellipse`、`Spline` 等曲线对象尝试 offset。

## 说明

- 闭合且能计算面积的对象通过面积变化判断内外。
- 开放曲线没有绝对内外概念，`W` 使用正距离，`N` 使用负距离。
