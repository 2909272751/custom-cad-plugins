# custom-cad-plugins v0.1.11

本次发布更新 `xrefpick` 到 `v0.1.7`。

## 更新

- `xrefpick-v0.1.7-autocad2021.zip`
  - 命令：`XREFPICK`
  - 新增 `H`：局部隐藏填充。
    - 外参：隐藏外参依赖填充图层，不影响本图同名图层。
    - 锁定普通块：隐藏块定义内 Hatch/Solid 对象，并记录恢复。
  - 新增 `U`：恢复局部隐藏的填充图层和填充对象。
  - 保留外参/锁定图层选择过滤、外参改色/恢复、日志命令 `XREFPICKLOG`。
