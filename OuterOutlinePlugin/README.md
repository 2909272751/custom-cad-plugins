# OuterOutlinePlugin

AutoCAD 2022 外轮廓线提取插件。

这个插件用于把楼层外轮廓从图纸里“抠出来”，生成一条闭合 `LWPOLYLINE`。插件本身不统计面积，生成闭合线后，你可以继续用现有面积统计工具统计 `PC_外轮廓` 图层。

## 适合解决什么问题

在装配式算量表里，“外轮廓线面积”通常需要一条干净的闭合外边界。手动用 `PL` 描外轮廓很慢，尤其遇到弧形外墙、梁线、墙线、多图层混合时容易漏线或走错线。

`OuterOutlinePlugin` 的目标是：

- 按你选择的图层样本自动提取候选线；
- 自动识别闭合外轮廓；
- 只输出一条更贴近外表面的闭合多段线；
- 输出到指定图层，方便后续面积统计。

## 命令

| 命令 | 作用 |
| --- | --- |
| `PCOUTLINE` | 提取楼层外轮廓闭合线 |
| `PCLOG` | 显示诊断日志路径 |

## 安装 / 加载

在 AutoCAD 命令行输入：

```text
NETLOAD
```

选择 DLL：

```text
C:\实用软件开发\OuterOutlinePlugin\dist\OuterOutlinePlugin-v0.1.5-autocad2022.dll
```

加载后输入：

```text
PCOUTLINE
```

## 新手操作流程

### 1. 框选楼层范围

命令行提示：

```text
框选需要提取外轮廓的楼层范围:
```

建议第一次只框选一个楼层或一个局部单体，不要直接框整张图。

### 2. 输入容差

命令行提示：

```text
端点吸附/断口容差，单位按当前图纸 <20>:
```

一般直接回车使用默认值 `20`。

如果图纸有小断口，可以试：

```text
30
50
```

### 3. 输入输出图层

命令行提示：

```text
生成外轮廓图层 <PC_外轮廓>:
```

一般直接回车，结果会生成到：

```text
PC_外轮廓
```

### 4. 连续点选外轮廓候选图层样本

命令行提示：

```text
点选边界图层样本对象继续添加，或按 Space/Enter 完成 [Remove]:
```

你要做的是：点选外轮廓可能会用到的线所在图层的“样本对象”。

比如可以点：

- 外墙外边线
- 梁外边线
- 墙边线
- 弧形外墙线
- 其他参与外轮廓闭合的边界线

每点一个对象，插件会读取这个对象所在图层，并立刻高亮框选区域内这个图层的所有对象。

如果高亮对象是你想参与计算的，就继续点下一个图层样本。

如果误选了图层：

```text
R
```

或输入：

```text
Remove
```

然后点一个该图层的对象，插件会移除这个图层并刷新高亮。

选择完成后按：

```text
Space
```

或：

```text
Enter
```

### 5. 自动生成外轮廓

插件会自动分析候选线网，生成闭合 `LWPOLYLINE`。

当前版本会固定只输出一条外轮廓，并在面积相同的候选轮廓里优先保留周长更短的那条，避免同时生成“内吃线”和“外表面线”。

## 选图层建议

优先选择真正形成外轮廓的图层：

- 外墙线
- 外边梁线
- 外轮廓弧线
- 外侧墙/梁的边线

尽量不要选择：

- 轴线
- 尺寸标注
- 文字
- 楼梯踏步线
- 内部分隔线
- 和外轮廓无关的填充线

如果选了太多内部图层，插件可能识别到多个候选闭合环。当前版本会尽量只保留更合理的外表面轮廓，但图层越干净，结果越稳定。

## 支持对象

当前支持：

- `LINE`
- `LWPOLYLINE` 直线段
- `LWPOLYLINE` 弧线段
- `ARC`
- `HATCH` 边界循环

暂未重点处理：

- 块内部线条
- 天正自定义对象内部几何
- 样条曲线
- 椭圆
- 三维非平面对象

## 失败或结果不对时

输入：

```text
PCLOG
```

查看日志路径。默认是：

```text
%TEMP%\PCOUTLINE.log
```

日志里重点看：

- `PCOUTLINE layers`：参与计算的图层
- `PCOUTLINE boundary entities`：参与计算的对象数量
- `PCOUTLINE raw segment count`：读取到的原始线段数量
- `PCOUTLINE faces`：识别到的闭合环数量
- `PCOUTLINE candidate`：候选外轮廓的评分、面积、周长、节点数

如果结果不对，把日志发给开发者，通常能判断是图层选多、线断开、对象类型不支持，还是候选筛选规则需要继续优化。
